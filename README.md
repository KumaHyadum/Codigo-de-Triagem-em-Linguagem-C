# Código de Triagem em Linguagem-C
Esse código representa uma pilha A e uma pilha B, na pilha A será colocado objetos que são maior que 15cm ou que sejam de plástico e na pilha B objetos que sejam de qualquer material mas que sejam menor que 15cm.
